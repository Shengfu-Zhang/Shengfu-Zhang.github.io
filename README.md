# shengfuzhang.github.io

Personal blog and artifact store by **Shengfu Zhang**.

Writing about AI, engineering systems, and how the two are reshaping the way we build software.

## Structure

```
posts/
└── YYYY-MM-slug/
    ├── blog.md           # Long-form blog post (Substack version)
    ├── linkedin.md       # LinkedIn post
    ├── twitter.md        # X/Twitter thread
    ├── xiaohongshu.md    # 小红书 post (Chinese)
    └── images/           # Cover image + diagrams
        ├── cover.png
        └── *.png
```

Each folder = one post. Self-contained. All platform variants live together.
